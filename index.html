BOBB (Be One Bouncing Ball)

Unity (android)- Puzzle game

BOOB (Be One Bouncing Ball) is a mobile game where you control a ball by moving your phone. Your objective will be to reach the finish line and find the hidden collectible in each level.

This game was made as a project for the Development of applications for mobile devices.

The goal was to use as many mobile sensors as possible. The game is controlled by the gyroscope, but we also implemented the possibility to take a picture and use it as a texture in the game and feedback to the player through vibration.


Roles:
  1. Game programmer
  2. UI programmer

Main menu with transitions.

You can take a picture and use it as your BOBB.

Rolling Beats

Unity (web)- Rythm game

Rolling beats is a rhythm video game where you play as Tina Rules, a skater whose dream is to become the first person to skate on the rings of Saturn.

This game was made as a project for the Web and social network games subject.

We decided to create a rhythm game with the goal of making it fun and enjoyable.

The entire team took care of every aspect of the game from start to finish, sticking to deadlines and managing the tasks that needed to be accomplished in each sprint.


Roles:
  1. UI artist and programmer
  2. Graphic designer

Play the game!

We choose a retrowave aesthetic for the game.

The player will earn coins to unlock new songs.

Each song has his own map and difficulty.

The space map was the first map we made.

There are 5 different maps, each one with their own UI assets.

The player can check how the game goes when the song is finished.

Sugar Escape

Unity (PC)- Adventure and puzzle game

In Sugar Escape you control Juangibre, an adorable cookie that must escape from his creator: a cute grandma who loves to bake. But the "realfood" is not going to make it easy, Agustincado won't let you go! Complete the puzzles that will help you escape and survive another day.

This game was made as a project for the Project management subject.

We had complete freedom to develop the game we wanted, but we had to organize ourselves as a team considering delivery dates and project management.


Roles:
  1. UI artist and programmer
  2. Localization

We keep the cookie vibes in all the elements of the game.

We implemented localization ENG-SPA.

A tutorial to guide the player and explain mechanics.

Simple interface but shows the necessary information.
